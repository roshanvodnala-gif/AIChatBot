# AI Chatbot in Java 🤖

This is a simple AI chatbot project that I built using Java.  
It is a basic rule-based chatbot that responds to user inputs using simple if-else conditions.

The chatbot runs in the console and keeps running until the user types an exit command.

---

## 💡 About the Project

I built this project to understand how basic chatbot systems work using programming logic.  
Instead of using machine learning or AI models, this chatbot works using predefined rules.

It helped me practice important Java concepts like loops, conditions, and user input handling.

---

## ⚙️ How it works

The chatbot takes input from the user and checks it using conditions:

- If the user types "hi" or "hello", it responds with a greeting  
- If the user asks "how are you", it gives a fixed response  
- If the user types "bye" or "exit", the program stops  
- For any other input, it replies that it doesn’t understand  

---

## 🧪 Example Output

You: hi  
Bot: Hello! How can I help you?

You: how are you  
Bot: I am fine. Thank you!

You: bye  
Bot: Goodbye!
